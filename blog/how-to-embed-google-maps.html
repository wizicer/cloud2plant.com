How to embed Google Maps

June 24, 2014

Surreal makes it easy to embed Google Maps into your pages. This article will show you how it’s done.

1. Get the embed code

Start by opening a new tab and going to Google Maps. Search for the location you want to show on the map. In this example, we’ve brought up Manchester, New Hampshire:

To get the embed code, select Share and embed map from the following menu. You’ll find it at the bottom-right of the maps screen:

Just copy the embed code from the resulting modal:

2. Embed the map into your page

Now that you’ve copied the embed code, head back into the editor. Select Code Snippet from the following menu:

Paste the embed code into the modal:

When you hit Submit, your map will be inserted into the page! (Note that the map will appear as an embedded object until you publish the page.)
