Prevent publishing conflicts

June 19, 2014

Ever wondered if someone else was editing the same page at the same time as you? Today we added a new feature that tells you just that. Here’s how it works.

When you’re viewing a site, you’ll see a subtle notification if someone else is editing a page:

You’ll see a more obvious alert if you go into the editor:

These alerts are designed to help you prevent publishing conflicts. That is, two people publishing the same page at nearly the same time. While no content is technically lost by doing this, it can be confusing and even frustrating to find that your changes were overwritten by another user.

Of course, if your changes are overwritten for some reason, you can retrieve them by going to Edit > View Revisions.
